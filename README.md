# Using tape and faucet to test React application

[tape](https://github.com/substack/tape)
[faucet](https://github.com/substack/faucet)
[enzyme](https://github.com/airbnb/enzyme)

## Usage

### Start
```sh
$npm run start
```

### Production build
```sh
$npm run build
```

### Test
```sh
$npm run test
```
